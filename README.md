> Node.js ¬∑ Next.js ¬∑ TypeScript ¬∑ BullMQ ¬∑ Redis ¬∑ Groq API ¬∑ Gmail API ¬∑ OAuth2

OAuth2 tokens scoped to minimum permissions, stored in Redis, never exposed client-side.
